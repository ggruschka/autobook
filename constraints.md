# Book Constraints

Everything in this file is a hard constraint. The agent must satisfy all of them in every iteration. Anything NOT specified here is up to the agent's creative interpretation.

## Required

- **Title**: (the book's title)
- **Genre**: (e.g., literary fiction, historical fiction, hard sci-fi, fantasy, memoir, popular science, self-help, etc.)
- **Target Audience**: (age group, assumed background, interests — this defines who the "Target Audience Reader" grader pretends to be)
- **Approximate Length**: (e.g., ~30,000 words, ~15 chapters, ~200 pages — give a rough target, not a hard limit)

## Optional

Fill in any that matter. Delete or leave blank any that don't — the agent will use its own judgment.

- **Writing Style**: (e.g., spare and minimalist, lush and descriptive, conversational, formal academic, etc.)
- **Point of View**: (e.g., first person, third person limited, omniscient, second person, multiple POV)
- **Setting**: (time period, location, world-building notes)
- **Themes**: (central themes to explore)
- **Tone**: (e.g., dark, humorous, hopeful, melancholic, satirical)
- **Structure**: (e.g., linear narrative, nonlinear, epistolary, frame story, anthology of linked stories)
- **Characters**: (any required characters, character types, or character constraints)
- **Moral/Ethical Guidelines**: (values the book should promote or avoid — e.g., "no glorification of violence", "emphasize empathy and resilience", "morally complex characters welcome, but consequences matter". The Moral Values grader evaluates strictly against what you write here)
- **Content Warnings**: (topics to avoid or handle with care)
- **Reference Works**: (books the result should feel like — "in the style of X", "similar to Y")
- **Language**: (language the book should be written in, default: English)

## Conditional Graders

Check which additional graders should evaluate the book:

- [ ] **Historian** — the book involves historical events or periods
- [ ] **Technical Reviewer** — the book covers technical or scientific subjects
